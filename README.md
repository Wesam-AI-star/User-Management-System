# 👥 User Management System (CRUD)

A simple Python console application to manage user profiles (Add, Display, Search, Update, and Delete) using Object-Oriented Programming (OOP) concepts.

## 🚀 Features
* **Add Member:** Creates a new user with a unique ID validation.
* **Display All:** Shows a formatted list of all registered users.
* **Advanced Search:** Find users by ID, First Name, or Status.
* **Update Data:** Modify First Name, Last Name, Password, or Status dynamically.
* **Delete User:** Completely removes a user and frees up their ID.

## 💻 How to Run
```bash
python user_management.py
